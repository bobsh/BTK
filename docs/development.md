# Local Development

The project uses "Rojo" to sync up the source with Roblox Studio,
make sure this is running and you setup synchronisation for a solid development workflow.

To run rojo:

    # make dev

## Tests

Run

    # make test

## Lint

Or for lint:

    # make lint
